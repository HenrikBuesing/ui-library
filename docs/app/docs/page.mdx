# Getting Started

<Steps>
  ### Installation

  ```sh npm2yarn
  npm install @hbuesing/ui-library
  ```
  
  **PeerDependencies**
  
  Note that [react](https://www.npmjs.com/package/react) and [react-dom](https://www.npmjs.com/package/react-dom)
  are peer dependencies. Make sure they are installed as well.
  
  ```sh npm2yarn
  npm install react react-dom
  ```

  ### importing the stylesheet

  ```tsx
  import "@hbuesing/ui-library/index.css";
  ```

  <Callout type="info">
    The stylesheet should be imported at the root of your project.
  </Callout>

  ### Start using components

  ```tsx
  import {Button} from "@hbuesing/ui-library";

  function Showcase() {
    return (
      <Button variant={'text'}>Text Button</Button>
    );
  }
  ```
</Steps>
