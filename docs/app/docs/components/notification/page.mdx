# Notification

A notification component is designed to deliver information or alerts to users in a clear and unobtrusive manner, such as system messages, reminders, or updates.

</details>

## Variants

The notification component offers two distinct variants: `filled` and `outlined`.

## Type options

The `type` property specifies the color scheme of the notification. Available options include `info`, `success`, `warning`, and `error`.

## User Interactivity

To add user interactivity use either the `onClose` or `action` prop. 

<Callout type={'warning'}>
  Note that these props are mutually exclusive. \
  If you just need a close button consider using the `onClose` prop, otherwise use the `action` prop instead.
</Callout>

### OnClose

If the `onClose` prop is set, a predefined close button will be added to the notification. 

<NotificationClose/>

<details>
  <summary>Expand code</summary>

  ```jsx
  <Notification variant={'filled'} type={'info'} onClose={() => {alert('close button clicked')}}>
    Notification with the predefined close button
  </Notification>

  <Notification variant={'outlined'} type={'info'} onClose={() => {alert('close button clicked')}}>
    Notification with the predefined close button
  </Notification>
  ```
</details>

### Action

The `action` prop allows you to add your own action to the notification

<NotificationAction/>

<details>
  <summary>Expand code</summary>

  ```jsx
  <Notification
    variant={'filled'}
    type={'info'}
    action={<Button variant={'text'} size={'small'} color={'#FFF'} onClick={() => {alert('action clicked')}}>Close</Button>}
  >
    Notification with a custom close button
  </Notification>

  <Notification
    variant={'outlined'}
    type={'info'}
    action={<Button variant={'text'} size={'small'} color={color} onClick={() => {alert('action clicked')}}>Close</Button>}
  >
    Notification with a custom close button
  </Notification>
  ```
</details>
