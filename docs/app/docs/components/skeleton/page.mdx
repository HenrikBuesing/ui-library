# Skeleton

A skeleton component provides a placeholder of the UI while data is loading, improving perceived responsiveness.

<DefaultSkeleton/>

<Detail>
  ```tsx
  <div className={'header'}>
    <Skeleton width={'2rem'} height={'2rem'} rounded={'full'}/>
    <Skeleton width={'15rem'}/>
  </div>

  <Skeleton height={'9rem'} rounded={'light'}/>
  <Skeleton width={'75%'} rounded={'light'}/>
  ```
</Detail>

## Size

Customize the skeleton dimensions with the `height` and `width` props.

<SkeletonSize/>

<Detail>
  ```tsx
  <Skeleton height={'7rem'} width={'12rem'}/>
  <Skeleton height={'10rem'} width={'5rem'}/>
  ```
</Detail>

## Pulse animation

By default, every Skeleton has a pulse animation. To disable it, use the `disableAnimation` prop.

<SkeletonAnim/>

<Detail>
  ```tsx
  <div className={'header'}>
    <Skeleton width={'2rem'} height={'2rem'} rounded={'full'} disableAnimation/>
    <Skeleton width={'15rem'} disableAnimation/>
  </div>

  <Skeleton height={'9rem'} rounded={'light'} disableAnimation/>
  <Skeleton width={'75%'} rounded={'light'} disableAnimation/>
  ```
</Detail>

<Callout type={'info'}>
  The default animation duration is 2.5 seconds. To customize it, override the CSS variable `--uil-pulse-anim`.
</Callout>

## Rounded

Use one of `light`, `medium` or `full` to customize the skeletons border radius.

<SkeletonRadius/>

<Detail>
  ```tsx
  <Skeleton height={'5rem'} width={'5rem'}/>
  <Skeleton height={'5rem'} width={'5rem'} rounded={'light'}/>
  <Skeleton height={'5rem'} width={'5rem'} rounded={'medium'}/>
  <Skeleton height={'5rem'} width={'5rem'} rounded={'full'}/>
  ```
</Detail>
