# Textarea

The textarea component allows users to enter and edit multi-line text.

<DefaultTextarea/>

<Detail>
  ```jsx
  <Textarea label={'Textarea'} rows={1}/>
  ```
</Detail>

The `textarea` and `input` components function similarly, with the key differences being that the `textarea` offers
only a single variant and does not support the `decorator` component.  
For more usage examples, please refer to the [input](/docs/components/input) documentation page.

## Resize

The textarea can be resized vertically, horizontally, both directions, or can be fixed without resizing.

<TextareaResize/>

<Detail>
  ```jsx
  <Textarea label={'Textarea'} rows={5} cols={50} resize={'none'}/>
  ```
</Detail>

<Callout>
  Use the props `rows` and `cols` to overwrite the default width and height.
</Callout>

## Validation

Use a combination of the error state and helpText prop to give users additional information.

<TextAreaValidate/>

<Detail>
  ```jsx
  <Textarea label={'Textarea'} rows={1} cols={75} resize={'vertical'} error 
            helpText={'Additional information about the textarea or error'} defaultValue={'Error'}/>
  ```
</Detail>
