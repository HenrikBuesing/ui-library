# Toast

A toast component is a small, temporary popup message used to provide brief feedback or notifications to users in an
unobtrusive manner.

<ToastDefault/>

<details>
  <summary>Expand code</summary>

  ```jsx
  <Button variant={'outlined'} onClick={() => setOpen(true)}>Show toast</Button>

  <Toast id={'simple-toast'} open={open} timeout={5000} closeCallback={() => {setOpen(false)}} alignment={{vertical: 'bottom', horizontal: 'left'}}>
    Toast message
  </Toast>
  ```
</details>

## Timeout

The `timeout` prop defines the duration (in ms) for which the toast will remain visible.

<ToastTimeout/>

<details>
  <summary>Expand code</summary>

  ```jsx
  <Input
    label={'Duration'}
    variant={'basic'}
    type={'number'}
    min={0}
    value={value}
    onChange={(event) => setValue(Number(event.target.value))}
    id={'timeout-duration'}
  />

  <Button variant={'outlined'} onClick={() => {setOpen(true)}}>
    Show toast
  </Button>

  <Toast id={'toast-timer'} open={open} timeout={value} closeCallback={() => {setOpen(false)}} alignment={{vertical: 'bottom', horizontal: 'left'}}>
    Timeout: {value}
  </Toast>
  ```
</details>

## Children

In addition to default HTML, you can also use the [Notification](/docs/components/notification) component.

<ToastContent/>

<details>
  <summary>Expand code</summary>

  <Toast id={'toast-content'} open={open} timeout={5000} alignment={{vertical: 'bottom', horizontal: 'left'}} closeCallback={() => {setOpen(false)}}>
    <Notification type={'success'} variant={'filled'}>Success notification</Notification>
  </Toast>

## Alignment

Use the `alignment` prop to override the default vertical (bottom) and horizontal (left) positioning of the toast.

<ToastAlignment/>

<details>
  <summary>Expand code</summary>

  ```jsx
  <span>Vertical alignment</span>
  <RadioGroup
    direction={'row'}
    name={'vertical'}
    selected={vertical}
    options={[{label: 'top', value: 'top'}, {label: 'center', value: 'center'}, {label: 'bottom', value: 'bottom'}]}
    onChange={(event) => setVertical(event.target.value)}
  />

  <span>Horizontal alignment</span>
  <RadioGroup
    direction={'row'}
    name={'horizontal'}
    selected={horizontal}
    options={[{label: 'left', value: 'left'}, {label: 'center', value: 'center'}, {label: 'right', value: 'right'}]}
    onChange={(event) => setHorizontal(event.target.value)}
  />

  <Button variant={'outlined'} onClick={() => {setOpen(true)}}>
    Show toast
  </Button>

  <Toast id={'toast-timer'} open={open} timeout={3000} closeCallback={() => {setOpen(false)}} alignment={{vertical: vertical, horizontal: horizontal}}>
    Toast with custom position
  </Toast>
  ```
</details>

<Callout type={'info'}>
  If you use the `ToastProvider` and set the alignment to bottom, the newest toast will be at the top of the stack.
</Callout>

## CloseCallback

The `closeCallabck` function will run after the set timeout. You can use this to reset the `open` state afterwards. 

## Actions

The `action` prop allows you to add additional user interactivity, such as including a close button.

<ToastActions/>

<details>
  <summary>Expand code</summary>

  ```jsx
  <Button variant={'outlined'} onClick={() => setOpen(true)}>Show toast</Button>

  <Toast
    id={'toast-action'}
    open={open}
    timeout={5000}
    closeCallback={() => {setOpen(false)}}
    alignment={{vertical: 'bottom', horizontal: 'left'}}
    action={<Button color={'success'} variant={'text'} size={'small'} onClick={() => {setOpen(false)}}>Close</Button>}
  >
    Wait for the timeout or close with the close button
  </Toast>
  ```
</details>

<Callout type={'info'}>
  When using the `Notification` component, you can instead utilize the `action` or `onClose` props for additional functionality.
</Callout>

<ToastActionsNotification/>

<details>
  <summary>Expand code</summary>

  ```jsx
  <Button variant={'outlined'} onClick={() => setOpen(true)}>Show toast</Button>

  <Toast id={'notify-toast'} open={open} timeout={5000} closeCallback={() => {setOpen(false)}} alignment={{vertical: 'bottom', horizontal: 'left'}}>
    <Notification type={'success'} variant={'filled'} onClose={() => {setOpen(false)}}>
      Wait for the timeout or close with the close button
    </Notification>
  </Toast>
  ```
</details>

## Stack

The `ToastProvider` component enables stacking multiple toasts on top of each other. For detailed usage instructions,
refer to the [ToastProvider](/docs/components/toast-provider) documentation.
