# BurgerMenu

The BurgerMenu component is a toggleable sidebar navigation menu that opens and closes via a burger button.

## Alignment

The `alignment` prop controls whether the sidebar menu should open on the left or right side of the screen.

<DefaultBurger align={'right'}/>
<DefaultBurger align={'left'}/>

<Detail>
  ```jsx
  <BurgerMenu alignment={'right'}>...</BurgerMenu>
  <BurgerMenu alignment={'left'}>...</BurgerMenu>
  ```
</Detail>

## Icon customisation

The close and open icons can be customized with the `openIcon` and `closeIcon` prop.

<IconBurger/>

<Detail>
  ```jsx
  <BurgerMenu alignment={'right'} openIcon={openIcon} closeIcon={closeIcon}>...</BurgerMenu>
  ```
</Detail>

## MenuHeader & MenuItem

The `MenuHeader` and `MenuItem` components are helper components for the burger menu. 
They accept `children` as props and are only useful in combination with the `BurgerMenu` component.

<Callout type={'info'}>
  Every example on this page uses these components.
</Callout>
