# DialogControls

This component is a utility component designed for the [`Dialog`](/docs/components/dialog) component.

<DialogControlsBasic/>

<Detail>
  ```jsx
  <Button variant={'outlined'} dark={dark} onClick={() => setIsVisible(true)}>Show Dialog</Button>

  <Dialog size={'small'} open={isVisible} onCancel={() => setIsVisible(false)} dark={dark}>
    <DialogTitle>Title</DialogTitle>

    <DialogContent>
      Dialog content
    </DialogContent>

    <DialogControls>
      <Button variant={'text'} onClick={() => {setIsVisible(false)}}>Confirm</Button>
      <Button variant={'text'} color={'error'} onClick={() => {setIsVisible(false)}}>Cancel</Button>
    </DialogControls>
  </Dialog>
  ```
</Detail>

<Callout type={'warning'}>
  Since `dialogContent` is a utility component, using it outside the `dialog` component will just render the content of the component:

  <Controls/>

  ```tsx
  <DialogControls>
    <Button variant={'filled'}>Button</Button>
  </DialogControls>
  ```
</Callout>

## Position

The position of the children can be either `start`, `end` or `space-between`.

<div className={styles.showcaseWrapper}>
  <DialogPos position={'start'} content={'Children of the dialog-controls will be left-aligned'}/>
  <DialogPos position={'end'} content={'Children of the dialog-controls will be right-aligned'}/>
  <DialogPos position={'space-between'} content={'Children of the dialog-controls will be positioned with space between them'}/>
</div>

<Detail>
  ```jsx
  <Button variant={'outlined'} dark={dark} onClick={() => setIsVisible(true)}>Show Dialog</Button>

  <Dialog size={'small'} open={isVisible} onCancel={() => setIsVisible(false)} dark={dark}>
    <DialogTitle>Title</DialogTitle>

    <DialogContent>
      Dialog content
    </DialogContent>

    <DialogControls position={'start'}> // one of 'start' | 'end' | 'space-between'
      <Button variant={'text'} onClick={() => {setIsVisible(false)}}>Confirm</Button>
      <Button variant={'text'} color={'error'} onClick={() => {setIsVisible(false)}}>Cancel</Button>
    </DialogControls>
  </Dialog>
  ```
</Detail>
