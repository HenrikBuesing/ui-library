# \<Icon />

## Assignable props
<AssignableProps
  data={[
    {name: 'type', value: '"svg" | "img"', required: true, definition: 'renders svg or img'},
    {name: 'src', value: 'string', required: true, definition: 'Source path'},
    {name: 'size', value: '"small" | "medium" | "large"', required: false, definition: 'size of the icon', default: 'medium'},
    {name: 'color', value: 'string', required: '', definition: 'Override color for svg icons'},
    {name: 'alt', value: 'string', required: '', definition: 'alt attribute for image icons'},
  ]}
/>

:::info
`alt` is required for icons with `type='img'`.
:::

## Example usage

### SVG Icon
<Icon type={'svg'} src={useBaseUrl('/img/example.svg#question')}/>

&NewLine;

```tsx
<Icon type={'svg'} src={'path/to/file.svg'}/>
```

### Sizes

Supported sizes are small (1rem), medium (2rem) and large (3rem). If you would like to set different sizes you can use the
`width` and `height` attributes.

<div className={'showcase'}>
  <Icon type={'svg'} src={useBaseUrl('/img/example.svg#question')} size={'small'}/>
  <Icon type={'svg'} src={useBaseUrl('/img/example.svg#question')} size={'medium'}/>
  <Icon type={'svg'} src={useBaseUrl('/img/example.svg#question')} size={'large'}/>
  <Icon type={'svg'} src={useBaseUrl('/img/example.svg#question')} height={64} width={64}/>
</div>

&NewLine;

```tsx
<Icon type={'svg'} src={'path/to/file.svg'} size={'small'}/>

<Icon type={'svg'} src={'path/to/file.svg'} size={'medium'}/>

<Icon type={'svg'} src={'path/to/file.svg'} size={'large'}/>

<Icon type={'svg'} src={'path/to/file.svg'} height={64} width={64}/>
```

### Custom color
<Icon type={'svg'} src={useBaseUrl('/img/example.svg#question')} color={'#0047CC'}/>

&NewLine;

```tsx
<Icon type={'svg'} src={'path/to/file.svg'} color={'#0047CC'}/>
```

:::note
This is only available for svg icons.
:::
