# \<CustomRadio />

## Assignable props
<AssignableProps
  data={[
    {name: 'options', value: 'RadioOption[]', required: true, definition: 'Radio options'},
    {name: 'value', value: 'string', required: true, definition: 'Current value'},
    {name: 'valueChanged', value: '(value: string) => void', required: true, definition: 'Update the current value'},
    {name: 'checkColor', value: 'string', required: false, definition: 'Set the check color'},
    {name: 'disabled', value: 'boolean', required: false, definition: 'disable all options'},
    {name: 'label', value: 'string', required: false, definition: 'Display a label for the radio options'},
  ]}
/>

## RadioOption
<AssignableProps
  data={[
    {name: 'label', value: 'string', required: true, definition: 'Label of the option'},
    {name: 'value', value: 'string', required: true, definition: 'value of the option'},
    {name: 'checked', value: 'boolean', required: false, definition: 'Check the option'},
    {name: 'disabled', value: 'boolean', required: false, definition: 'Disable the option'},
  ]}
/>

:::info
If one of your options has the `checked` property, the current value will be automatically set to the option's value. \
Note that if you provide more than one option with the `checked` property, the last one will be selected.
:::

## Example usage

### Radio options without a label
<RadioExample/>

&NewLine;

```tsx
<CustomRadio
  options={options}
  value={value}
  valueChanged={setValue}
/>
```

```tsx title="Radio options"
const options: RadioOption[] = [
  {value: 'option-a', label: 'option-a', checked: true},
  {value: 'option-b', label: 'option-b'},
  {value: 'option-c', label: 'option-c'},
  {value: 'option-d', label: 'option-d', disabled: true},
];
```
