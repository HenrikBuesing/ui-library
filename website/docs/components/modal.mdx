:::info
  The modal body can also be overridden by your own HTML.
:::

## Notification

### No theme
<NoTheme/>

&NewLine;

````tsx
<Modal
  type={'notification'}
  title={'No theme'}
  message={'This is a notification without a theme'}
  action={() => setModalVis(false)}
  buttonLabel={'close'}
/>
````

### Separate message in paragraphs
If you would like to separate the message into different paragraphs, use an array of strings.

<MultiLineModal/>

&NewLine;

```tsx
<Modal
  type={'notification'}
  title={'Multiple messages'}
  message={['first string', 'this is a second text', 'Each string is wrapped in a paragraph']}
  action={() => setModalVis(false)}
  buttonLabel={'close'}
  theme={'success'}
/>
```

### Timeout
The `action` function runs automatically after the set `timeout`.

<CallbackModal/>

&NewLine;

```tsx
<Modal
  type={'notification'}
  title={'Modal with timeout'}
  message={'close after after 3 second timeout'}
  action={alertTimeout}
  buttonLabel={'close'}
  theme={'success'}
  timeout={3000}
/>
```

### Custom modal content
You can provide custom HTMl for both `Notification` & `Question` instead of providing certain props.

<CustomBody/>

&NewLine;

```tsx
<Modal type={'notification'} title={'custom modal content'} theme={'success'}>
  <div>
    <p>paragraph text</p>
    <div>you can also insert <a href={'/'}>links</a></div>
  </div>

  <div>
    <Button label={'close modal'} buttonType={'text'} onClick={() => setModalVis(false)}/>
  </div>
</Modal>
```

### Dark version

```tsx
/>
```

### Playground
```tsx live
function Playground() {
  const [modalVis, setModalVis] = useState(false);

  return (
    <>
      <Button buttonType={'secondary'} label={'show Modal'} onClick={() => setModalVis(true)} />

      {modalVis &&
        <Modal
          type={'notification'}
          title={'Modal title'}
          message={'Modal message'}
          action={() => setModalVis(false)}
          buttonLabel={'close label'}
          theme={'success'}
        />
      }
    </>
  );
}
```
