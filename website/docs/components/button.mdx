# \<Button />

## Assignable props
<AssignableProps
  data={[
    {name: 'label', value: 'string', required: true, definition: 'Button label'},
    {name: 'disabled', value: 'boolean', required: false, definition: 'Enable or disable the button', default: 'false'},
    {name: 'small', value: 'boolean', required: false, definition: 'Decreases the size of the button', default: 'false'},
    {name: 'theme', value: 'HEX code | "success" | "warning" | "error"', required: false, definition: 'Set the button theme', default: 'undefined'},
    {name: 'HTML button props', value: '', required: false, definition: 'HTML button attributes, e.g. "onClick"'},
  ]}
/>

:::note[Providing a custom color for the theme prop]
The provided HEX code must be 7 (including '#') characters long, e.g. '#1A2B3C'
:::

The font color (either black or white) will be chosen based on the provided theme. \
Not providing a theme will generate the default styled button.

## Example usage
Buttons can be rendered with different styles.
