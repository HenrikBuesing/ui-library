# \<Button />

## Assignable props
<AssignableProps
  data={[
    {name: 'buttonType', value: '"primary" | "secondary" | "outline" | "text"', required: true, definition: 'Button style'},
    {name: 'label', value: 'string', required: true, definition: 'Button label'},
    {name: 'dark', value: 'boolean', required: false, definition: 'Enables dark mode', default: 'false'},
    {name: 'disabled', value: 'boolean', required: false, definition: 'Enable or disable the button', default: 'false'},
    {name: 'size', value: '"small" | "medium" | "large"', required: false, definition: 'Set the size of the button', default: 'medium'},
    {name: 'theme', value: 'HEX | "success" | "warning" | "error"', required: '', definition: 'Set the button theme'},
    {name: 'HTML button props', value: '', required: false, definition: 'HTML button attributes, e.g. "onClick"'},
  ]}
/>

:::warning[When to use a theme]
The theme prop must only be used for button types `primary` & `outline`.
:::


If you want to use a custom color note that the provided `HEX` code must be a valid `HEX Color` with a maximum length of 6 characters (e.g. `#0F0` or `#00FF00`). \
The font color (either black or white) will be chosen based on the provided theme.

## Example usage
Buttons can be rendered with different styles and come with a light and dark version.

### Primary button

<div className={'showcase'} style={{backgroundColor:'#FFFFFF'}}>
  <Button buttonType={'primary'} label={'Success'} theme={'success'}/>
  <Button buttonType={'primary'} label={'Warning'} theme={'warning'}/>
  <Button buttonType={'primary'} label={'Error'} theme={'error'}/>
  <Button buttonType={'primary'} label={'custom color'} theme={'#b554e6'}/>
</div>

