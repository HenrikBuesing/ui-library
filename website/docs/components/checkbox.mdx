# \<Checkbox />

## Assignable props
<AssignableProps
  data={[
    {name: 'checked', value: 'boolean', required: true, definition: 'Current value'},
    {name: 'toggleCheck', value: '(value: boolean) => void', required: true, definition: 'Check or uncheck'},
    {name: 'color', value: 'string', required: false, definition: 'Set the check color'},
    {name: 'dark', value: 'boolean', required: false, definition: 'enables dark version', default: 'false'},
    {name: 'label', value: 'string', required: false, definition: 'Label for the checkbox'},
    {name: 'children', value: 'ReactNode', required: false, definition: 'Use custom content instead of the label'},
    {name: 'HTML input props', value: '', required: false, definition: 'HTML input attributes, e.g. "id"'},
  ]}
/>

:::info
You can only use either the label property or children.
:::

## Example usage

### Checkbox with the label property
<CheckboxLabel/>

&NewLine;

```tsx
<Checkbox
  checked={value}
  toggleCheck={setValue}
  label={'light checkbox'}
/>
```

<CheckboxLabelDark/>

&NewLine;

```tsx
<Checkbox
  checked={value}
  toggleCheck={setValue}
  label={'dark checkbox'}
  dark={true}
/>
```

### Custom checked color
<CheckboxColor/>

&NewLine;

```tsx
<Checkbox
  checked={value}
  toggleCheck={setValue}
  label={'Custom check Color'}
  checkColor={'red'}
  name={'check-with-custom-color'}
/>
```

<CheckboxColorDark/>

&NewLine;

```tsx
<Checkbox
  checked={value}
  toggleCheck={setValue}
  label={'Custom check Color'}
  checkColor={'red'}
  dark={true}
/>
```

### Custom content
<CheckboxBody/>

&NewLine;

```tsx
<Checkbox checked={value} toggleCheck={setValue}>
  <span>You can provide html elements as a label alternative with <a href={"..."}>links</a></span>
</Checkbox>
```

<CheckboxBodyDark/>

&NewLine;

```tsx
<Checkbox checked={value} toggleCheck={setValue} dark={true}>
  <span>You can provide html elements as a label alternative with <a href={"..."}>links</a></span>
</Checkbox>
```

### Disabled checkbox
<CheckboxDisabled/>

&NewLine;

```tsx
<Checkbox checked={value} toggleCheck={setValue} disabled={true} label={'disabled light'}/>
```

<CheckboxDisabledDark/>

&NewLine;

```tsx
<Checkbox checked={value} toggleCheck={setValue} dark={true} disabled={true} label={'disabled dark'}/>
```
