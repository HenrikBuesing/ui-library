# \<CustomCheckBox />

## Assignable props
<AssignableProps
  data={[
    {name: 'checked', value: 'boolean', required: true, definition: 'Current value'},
    {name: 'toggleCheck', value: '(value: boolean) => void', required: true, definition: 'Check or uncheck'},
    {name: 'checkColor', value: 'string', required: false, definition: 'Set the check color'},
    {name: 'label', value: 'string', required: false, definition: 'Label for the checkbox'},
    {name: 'children', value: 'ReactNode', required: false, definition: 'Use custom content instead of the label'},
    {name: 'HTML input props', value: '', required: false, definition: 'HTML input attributes, e.g. "name"'},
  ]}
/>

:::note
You can only use either the label prop or children. If both are provided children will be used.
:::

## Example usage

### Label property
<CheckboxLabel/>

&NewLine;

```tsx
<CustomCheckBox
  checked={value}
  toggleCheck={setValue}
  label={'label property'}
  name={'check-with-label'}
/>
```

### Custom checked color
<CheckboxColor/>

&NewLine;

```tsx
<CustomCheckBox
  checked={value}
  toggleCheck={setValue}
  label={'Custom check Color'}
  checkColor={'red'}
  name={'check-with-custom-color'}
/>
```

### Custom content
<CheckboxBody/>

&NewLine;

```tsx
<CustomCheckBox checked={value} toggleCheck={setValue} name={'check-with-custom-label'}>
  <div>
    <span>You can provide html elements as a label alternative</span>
  </div>
</CustomCheckBox>
```
